# Next.js 16 Application

A modern web application built with **Next.js 16**, leveraging the latest features for optimal performance and user experience.

## Key Features

- **Server-Side Rendering (SSR)** - Enhanced performance with Next.js 16's improved SSR capabilities
- **Advanced Caching** - Utilizing Next.js 16's cache optimization for faster page loads
- **React 19** - Built with the latest React features and improvements
- **Tailwind CSS v4** - Modern styling with the latest Tailwind CSS version
- **TypeScript** - Full type safety throughout the application

## Tech Stack

- Next.js 16.1.0
- React 19.2.3
- Tailwind CSS v4
- TypeScript 5

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the application.

## Project Structure

```
app/          # Next.js App Router
  ├── layout.tsx
  ├── page.tsx
  └── globals.css
public/       # Static assets
```
